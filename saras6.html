SARAS

Simulator for AeRospace ApplicationS

Test bed to simulate six degrees of freedom motion of an aerospace vehicle. SARAS simulates translational and rotational motion of the vehicle. The subsystems are mathematically modeled to realistically represent the vehicle and environment.

Core Capabilities

Multifaceted Software

Mathematical models of dynamics and environment

External Integration

Integration with range of external tools and sensors

Real-time Processing

Processing of multiple sensor data and display

Force & Moment Computation

Computes all the forces and moments acting on the vehicle

Subsystem Simulation

Simulates subsystems independently and integrated mode

Off-Nominal Conditions

Simulates the vehicle dynamics in off-nominal conditions

Configuration & Analysis

Configurable as design/analysis tool

Monte Carlo simulations, Range/Mission Safety analysis and visibility studies

Configurable for MILS, OILS, HILS

Simulation Modes

Model in Loop Simulation

(MILS)
Mathematical formulation and Design Verification tool
  • Navigation Guidance and Control (NGC) models
  • Attitude and Orbit Control System (AOCS) Models
  • Sensor and actuator mathematical models

Onboard in Loop Simulation

(OILS)
Flight Software validation tool
  • NGC/AOCS Software in target processor
  • Sensors and Actuator models
  • Communication via Ethernet

Hardware in Loop Simulation

(HILS)
NGC/AOCS System Test Bed
  • NGC/AOCS Software in target processor
  • Sensors and Actuators Hardware in the loop
  • Communication via Ethernet
